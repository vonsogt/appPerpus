# Aplikasi Perpustakaan
Aplikasi elektronik perpustakaan cross-platform

## Instalasi
1. Klon/Unduh projek ini

   `git clone https://github.com/vonsogt/appPerpus.git`
2. Pindah ke direktori

   `cd appPerpus`
3. Install modul

   `npm install`
4. Jalankan aplikasi

   `ionic serve`
